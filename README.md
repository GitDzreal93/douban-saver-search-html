# 豆瓣帖子保存器（可搜索 HTML 版）

一个 Chrome 扩展，用来把豆瓣帖子保存成自包含的本地 HTML，并提供一个浏览器内的本地搜索页，方便在 Windows 或 Mac 上直接搜索已保存内容。

## 功能

- 保存豆瓣帖子为本地 HTML
- 支持按标题、昵称、回复内容搜索
- 提供本地搜索页，手动选择帖子文件夹后即可索引
- 提示用户在保存新帖子后手动刷新索引

## 使用方式

1. 安装 Chrome 扩展
2. 在豆瓣帖子页面点击保存
3. 打开扩展里的“搜索已保存帖子”
4. 选择保存 HTML 的文件夹
5. 输入关键词搜索

## 说明

- 搜索功能完全在本地运行
- 不会把帖子内容上传到服务器
